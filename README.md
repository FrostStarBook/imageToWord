# imageToWord

> 图片转换字符(注释详细)

### 运行环境

- python3

```
sudo pip3 install pillow
```

### 原理

> 字符画是一系列字符的组合，我们可以把字符看作是比较大块的像素，一个字符能表现一种颜色（彩色图片对应灰度图像），字符的种类越多，可以表现的颜色也越多，图片也会更有层次感。
